# Prueba de Lentes Virtual

Simula cómo te quedan lentes usando la cámara de tu iPhone directamente desde el navegador.

## Cómo usar

1. Abre la página en tu iPhone
2. Permite el acceso a la cámara
3. Selecciona unos lentes del catálogo
4. Mira cómo te quedan en tiempo real
5. Toma una foto con el botón "Capturar"

## Deploy en GitHub Pages

1. Sube el código a un repositorio en GitHub
2. Ve a **Settings > Pages**
3. En **Source** selecciona la rama `main`
4. Abre la URL que te genere GitHub

## Tecnologías

- **JeelizFaceFilter** - Detección facial en tiempo real
- **Canvas 2D** - Overlay de lentes
- **WebGL** - Renderizado de video
- **HTML/CSS/JS** - Sin frameworks, vanilla
